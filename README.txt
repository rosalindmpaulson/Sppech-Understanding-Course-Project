To compile this paper to PDF in CVPR LaTeX format:

1. Upload `ser_cvpr.tex` to Overleaf (https://overleaf.com) or a local LaTeX editor.
2. Ensure the `cvpr.sty` and required CVPR template files are available (can be downloaded from the official CVPR author kit).
3. Click 'Recompile' in Overleaf or run `pdflatex ser_cvpr.tex` locally.
4. The output will be `ser_cvpr.pdf`.

If using Overleaf, choose the CVPR2024 or CVPR2023 template and replace `main.tex` with `ser_cvpr.tex`.
